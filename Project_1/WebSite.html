HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It uses tags to structure and format content, defining elements like headings, paragraphs, links, and images.

CSS

CSS (Cascading Style Sheets) is a styling language used for web design. It controls the layout, appearance, and presentation of HTML elements on web pages, enhancing visual aesthetics.

JavaScript

JavaScript (JS) is a versatile, high-level programming language used for web development. It enhances interactivity, dynamically modifying HTML and handling user interactions in browsers, creating rich web applications.
